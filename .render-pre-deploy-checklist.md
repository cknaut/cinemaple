# Pre-deploy checklist (Render)

Use this **before** you run **New → Blueprint** on Render.

## 1. Repo is on GitHub

- [ ] Project is pushed to a GitHub repo (e.g. `your-username/cinemaple`).
- [ ] Branch is `main` or `master` (whichever you use).

## 2. Required env values ready (to paste in Render after Blueprint)

Blueprint will create the service and DB. Then you must add these in **Environment**:

| Variable | Where to get it |
|----------|-----------------|
| `RECAPTCHA_SECRET_KEY` | [Google reCAPTCHA](https://www.google.com/recaptcha/admin) → create v2 site → Secret key |
| `RECAPTCHA_SITE_KEY` | Same → Site key |
| `EMAIL_VERIFICATION_SECRET_SALT` | Run `python scripts/generate_salts.py` → first value |
| `PW_RESET_SECRET_SALT` | Same script → second value |
| `REV_USER_ACCESS_SECRET_SALT` | Same script → third value |
| `TMDB_API_KEY` | [TMDb API](https://www.themoviedb.org/settings/api) → Request API Key (free) |

Optional (can leave unset): all `MAILCHIMP_*` and `MAILGUN_*`.

## 3. After first deploy

- [ ] Open the service URL (e.g. `https://cinemaple-xxxx.onrender.com`) and confirm the app loads.
- [ ] Add custom domain in Render (**Settings → Custom Domains**): `www.cinemaple.com`.
- [ ] In Cloudflare (**DNS**): CNAME `www` → Render hostname (e.g. `cinemaple-xxxx.onrender.com`).
